# Formula 1 Wikipedia

A full-stack web application built with React and Python, designed to organize and browse Formula 1 data — drivers, teams, and more. 
This project was developed as a personal challenge to level up full-stack development skills.

## ✨ Features

- **Driver Catalog** — Browse all F1 drivers with detailed info for each.
- **Team Catalog** — Explore every constructor with key stats and history.
- **Admin Panel** — Add new drivers and teams directly through the UI.
- **Full-Stack Architecture** — React frontend communicating with a Python backend API.

- **Frontend:** React

## ⚙️ How to Run It

You can easily download and run this project locally on your machine.

**Prerequisite:** Make sure you have [Node.js](https://nodejs.org/) and [Python](https://www.python.org/downloads/) installed.

**1. Clone the repository**

```bash
git clone https://github.com/IzabelaxKo/F1-wiki.git
```

**2. Open the project's frontend folder**

```bash
cd F1-wiki
```

**3. Install dependencies**

```bash
npm install
```

**4. Start the frontend**

```bash
npm run dev
```

**5. Go to backend**

```bash
cd ..\backend
```

**6. Install Python requirements**

```bash
pip install -r requirements.txt
```

**7. Start the backend**

```bash
python main.py
```
